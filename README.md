# Project Name : cati

# Info:
Show images at terminal true colors/terminal colors. PNG




# Usage:
```
$> make
$> ./cati image.png
```

# options:
```
$> ./cati image.png -t  - true color
$> ./cati image.png -o  - original size of image
```
